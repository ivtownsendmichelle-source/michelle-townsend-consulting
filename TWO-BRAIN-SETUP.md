# Two-Brain System

Two separate knowledge management folders. Two separate Claude Code sessions. They never see each other.

## MTC-Brain
Michelle Townsend Consulting. Clients, revenue, tools, builds.
- owner-inbox/ . Finished work for review
- team-inbox/ . Drop notes, transcripts, screenshots for processing
- clients/ . One file per client
- projects/ . One file per project
- daily/ . Daily itinerary files (YYYY-MM-DD.md)
- resources/ . Reference docs and templates
- viewer.html . Dark-mode dashboard (open in any browser)

## OFMA-Brain
Oregon Farmers Markets Association. Meetings, policy, documents.
- owner-inbox/ . Finished work for review
- team-inbox/ . Drop items for processing
- meetings/ . One file per meeting
- policy/ . AI policy drafts, governance docs
- documents/ . General OFMA documents
- daily/ . Daily notes for OFMA work

## How to Use
1. Open Claude Code pointed at MTC-Brain for consulting work
2. Open a separate Claude Code session pointed at OFMA-Brain for OFMA work
3. Each session reads its own CLAUDE.md and knows what to do
4. Drop files in team-inbox/ and ask Claude to process them
